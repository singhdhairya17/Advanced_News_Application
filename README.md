# Advanced News Application (Citizen Post)

React news reader: global and India editions, categories, search, and dark mode. Headlines via [NewsAPI](https://newsapi.org).

## Run locally

From this folder (repo root):

```bash
npm install
npm start
```

Then open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```

Optional API key: create a `.env` file with `REACT_APP_NEWS_API_KEY=your_key`.

## Project layout

- `src/` — React app (`components`, `hooks`, `services`, `constants`, `styles`)
- `public/` — static assets and `index.html`

If an empty `Dhairya News_Application` folder remains after a move, close the editor, delete that folder in File Explorer, and reopen this repository root.
